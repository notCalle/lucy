Class Queue

Basic queue class, supporting both lifo and fifo extraction modes.

See also:

    List

Usage:

    Queue = require'lucy.queue'
q = Queue{}
q:push(1) q:push(2) q:push(3)
1 == q:pull()
3 == q:pop()

Methods

Queue:push (item) Push an item onto the queue.
Queue:pop () Pop item from the tail of the queue
Queue:pull () Pull item from the head of the queue
