Class Mixin

Base class for mixins.

See also:

  • Class
  • Object

Usage:

    Object = require'lucy.object'
Mixin = require'lucy.mixin'
MyMixin = Mixin'MyMixin'
MyObject = Object'MyObject'+MyMixin

Class metamethods

Mixin:__add (mixin) Instantiate mixin in receiver.
Mixin:__bor (other) Merge mixin into receiver.
Mixin:__mod (map) Filter a mixin, renaming methods to enable overrides.
