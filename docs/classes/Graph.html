Class Graph

Basic directed graph mixin, with no edge restrictions

See also:

    Object

Usage:

    MyGraph=Object'MyGraph'+Graph

v1 = MyGraph{}             (v1)
v2 = MyGraph{}^v1          (v2)->(v1)
v3 = MyGraph{}^{v1,v2}     (v3)->(v2)->(v1)
                               `--------^

Metamethods

Graph:__pow (head, tail)
Add an edge between vertices. Either head or tail, but not both may be a table of Graph objects. In that case edges are added to/from all objects in the table.

N.B! call signature is actually Graph.__pow(head,tail), but ldoc.

Parameters:

  • head Graph or {Graph,...}
  • tail Graph or {Graph,...}

Returns:

    Graph or {Graph,...} head for method chaining

Usage:

    v1^{v2,v3}^v4  <=>  v1^v2 ; v1^v3 ; v2^v4 ; v3^v4

Graph:edges_bfs (select)
Return an iterator function that walks the edges of a graph, breadth first

If called with a function, that function must return truthy for an edge to be walked, for example to detect loops and avoid infinite iteration, or implementing a state machine.

The iterator yields two values: head,tail for the next selected edge.

Parameters:

  • select optional func function that selects edges to walk

Returns:

    func 'for' loop iterator
